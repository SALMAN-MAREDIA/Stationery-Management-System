# Hotel Oasis Management System

A full-stack hotel management website for **Hotel Oasis**, Mumbai. Built with Node.js, Express, EJS, and SQLite.

## Features

- **Home Page** – Hero section, amenities showcase, room previews, and tariff table
- **Rooms Page** – All available rooms with pricing and amenities
- **Gallery Page** – Filterable gallery with categories (Rooms, Lobby, Exterior, Dining)
- **About Us** – Hotel history, features, and timeline
- **Contact Us** – Contact form with details, stored in database
- **Booking System** – Multi-step booking form with room selection and date picker
- **Payment Gateway** – Simulated payment processing (Card, UPI, Net Banking)
- **Admin Login** – Secure admin authentication (default: admin / admin123)
- **Admin Dashboard** – View bookings, rooms, messages with statistics
- **WhatsApp Integration** – Floating WhatsApp icon for quick contact

## Tech Stack

- **Backend**: Node.js, Express 5
- **Templating**: EJS
- **Database**: SQLite (via better-sqlite3)
- **Auth**: bcryptjs, express-session
- **Frontend**: HTML5, CSS3 (custom), JavaScript (vanilla)

## Quick Start

```bash
npm install
npm start
```

Open http://localhost:3000 in your browser.
